# 一个简易的基于 React + Ant Design Pro 的博客后台管理系统

## 前言

Ant Design Pro 是基于 Ant Design 和 umi 的封装的一整套企业级中后台前端/设计解决方案

React 是一个用于构建用户界面的 JavaScript 库。其中主要使用了 React Hooks + 函数组件

## 安装

```shell
// 安装依赖包
npm i
// 运行
npm start
```

打开浏览器，输入 http://localhost:8000
